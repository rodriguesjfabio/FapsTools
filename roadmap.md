# Tools Roadmap

This document lists common utility tools found on the internet, grouped by category, with a brief description of each.

## Password & Security

- **Password Strength Checker** - Analyze a password and provide feedback on its strength based on length, character variety, and common patterns.
- **Password Generator** - Create random passwords with configurable length and character sets (letters, numbers, symbols).
- **Password Leak Checker** - Verify if a password appears in known data breaches.

## Text & Data Manipulation

- **Base64 Encoder/Decoder** - Convert data to and from Base64 encoding.
- **JSON Formatter & Validator** - Format, minify, and validate JSON input.
- **Regex Tester** - Enter a regular expression and a test string to see matches and groups.
- **Case Converter** - Convert text between upper case, lower case, title case, etc.
- **URL Encoder/Decoder** - Encode or decode URL components.

## Conversion Tools

- **Color Converter** - Convert between color formats: HEX, RGB, HSL, CMYK.
- **Number Converter** - Convert numbers between decimal, binary, hexadecimal, and octal.
- **Unit Converter** - Convert between various measurement units (length, weight, volume, temperature).
- **Time Zone Converter** - Convert dates/times between time zones.

## Development Helpers

- **QR Code Generator** - Create QR codes from text or URLs.
- **UUID Generator** - Generate random UUIDs (v1, v4).
- **Cron Expression Builder** - Generate CRON expressions using a calendar interface.
- **Lorem Ipsum Generator** - Generate placeholder text in various lengths.

## Networking & Web

- **IP Lookup** - Fetch geolocation and ISP information for an IP address.
- **HTTP Header Viewer** - Display HTTP headers returned by a URL.
- **User-Agent Parser** - Decode user-agent strings to identify browser/device.

## Media & Files

- **Image to Base64 Encoder** - Convert uploaded images to Base64 strings.
- **Video to Gif Converter** - Extract a segment of a video and convert to GIF.
- **File Hash Generator** - Calculate MD5, SHA-1, SHA-256 hashes of uploaded files.

## Productivity

- **Calendar to CRON** - Generate CRON expressions by selecting dates on a calendar.
- **Todo List / Notes** - Simple interface to track tasks or notes.

## SEO & Analytics

- **Domain Authority Checker** - Estimate a domain's authority score using public APIs.
- **Backlink Checker** - List backlinks pointing to a given URL.
- **PageSpeed Insights** - Analyze page load performance with suggestions.
- **Keyword Density Analyzer** - Count keyword occurrences in text.

## Math & Calculators

- **Mortgage Calculator** - Compute monthly payments for loans.
- **BMI Calculator** - Body mass index from height and weight.
- **Unit Price Calculator** - Compare prices per unit quantity.
- **Tip & Split Calculator** - Split a bill and calculate gratuity.
- **Base Conversion Advanced** - Support for arbitrary bases and fractions.
- **Percentage Calculator** - Increase, decrease or percentage of numbers.
- **Equation Solver** - Solve linear/quadratic equations step-by-step.

## Image & Media Tools

- **Image Compressor** - Reduce image file size with quality options.
- **Image Resizer** - Change image dimensions online.
- **SVG Optimizer** - Minify and clean SVG code.
- **Video Metadata Viewer** - Display codecs, resolution, duration.
- **Audio Converter** - Convert between MP3, WAV, AAC, OGG, etc.
- **YouTube Audio Downloader** - Extract audio from YouTube videos (MP3, AAC, OGG and more) and entire playlists.
- **GIF Maker** - Create animated GIFs from multiple images or video.

## Code & Development Utilities

- **HTML/CSS/JS Minifier** - Minify static web files.
- **CSS Prefixer** - Add vendor prefixes for cross-browser support.
- **JavaScript Obfuscator** - Obfuscate JS code for deployment.
- **SQL Formatter & Validator** - Pretty-print and check SQL queries.
- **XML Formatter & Validator** - Indent XML and verify well-formedness.
- **XPath Tester** - Evaluate XPath expressions against XML.
- **JSON to CSV/CSV to JSON Converter** - Transform between formats.
- **API Requester (similar to Postman light)** - Construct and execute HTTP requests.
- **JWT Encoder/Decoder** - Encode payloads and decode JSON Web Tokens.
- **CORS Checker** - Test cross-origin resource sharing headers.

## Security & Privacy

- **Hash Cracker (Dictionary)** - Attempt to brute-force hashes using wordlists.
- **Entropy Calculator** - Estimate randomness level of strings.
- **SSL Certificate Checker** - Show expiration and validity of TLS certs.
- **Port Scanner** - Check open ports on a host.

## Finance & Business

- **Currency Converter** - Convert using real-time exchange rates.
- **Loan Amortization Schedule** - Detailed payment breakdown for loans.
- **Stock Symbol Lookup** - Fetch company names and prices.
- **Invoice Generator** - Create printable invoices with line items.

## Productivity & Office

- **PDF Split/Merge/Compress** - Manipulate PDF documents.
- **Word Counter & Reading Time Estimator** - Count words and estimate duration.
- **Markdown Previewer** - Render Markdown to HTML in real time.
- **Cryptogram Solver** - Help decrypt simple substitution ciphers.

## Miscellaneous

- **Color Palette Generator** - Create complementary/analogous color palettes.
- **Random Word/Name Generator** - Produce random words or names based on criteria.
- **Unit Tester** - A playground to write and run small code snippets (JS, Python).
- **Emoji Translator** - Replace words with appropriate emojis.
- **Age Calculator** - Compute age from birthday.
- **Date Difference Calculator** - Days between two dates.
- **QR Code Scanner (via webcam)** - Decode QR codes using device camera.
- **Lorem Ipsum Variants** - Latin, Bacon ipsum, corporate ipsum.
- **Text Diff Tool** - Compare two pieces of text and highlight changes.
- **Regex Library/Reference** - Examples and syntax cheat-sheet.
